# Monitoring

Will be used to create an invoice item. The price must be in cents: 2000 for $20.

## Example Usage

```typescript
import { Monitoring } from "@simplesagar/vercel/models/authuser.js";

let value: Monitoring = {
  price: 4413.21,
  quantity: 452.34,
  hidden: false,
};
```
