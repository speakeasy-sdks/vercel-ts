# GetDomainsServiceType

The type of service the domain is handled by. `external` if the DNS is externally handled, `zeit.world` if handled with Vercel, or `na` if the service is not available.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ZeitWorld` | zeit.world  |
| `External`  | external    |
| `Na`        | na          |
