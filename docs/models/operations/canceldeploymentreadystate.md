| -------------- | -------------- |
| `Initializing` | INITIALIZING   |
| `Building`     | BUILDING       |
| `Ready`        | READY          |
| `Error`        | ERROR          |
| `Queued`       | QUEUED         |
| `Canceled`     | CANCELED       |
