# GitSource

Defines the Git Repository source to be deployed. This property can not be used in combination with `files`.

## Example Usage

```typescript
import { GitSource } from "@simplesagar/vercel/models/createdeploymentop.js";

let value: GitSource = {
    org: "<value>",
    ref: "<value>",
    repo: "<value>",
    type: "github",
};
```
