# Authentication
(*authentication*)

### Available Operations

* [listTokens](#listtokens) - List Auth Tokens
* [createToken](#createtoken) - Create an Auth Token

## listTokens

Retrieve a list of the current User's authentication tokens.

### Example Usage

```typescript
import { Vercel } from "vercel";

const vercel = new Vercel({
  security: {
    bearerToken: "<YOUR_BEARER_TOKEN_HERE>",
  },
});

async function run() {
  const result = await vercel.authentication.listTokens();

  // Handle the result
  console.log(result)
}

run();
```

## createToken

### Example Usage

```typescript

const vercel = new Vercel({
  security: {
    bearerToken: "<YOUR_BEARER_TOKEN_HERE>",
  },
});

async function run() {
  const result = await vercel.authentication.createToken();

  // Handle the result
  console.log(result)
```

## verifyToken

Verify the user accepted the login request and get a authentication token. The user email address and the token received after requesting the login must be added to the URL as a query string with the names `email` and `token`.

### Example Usage

```typescript

const vercel = new Vercel();

async function run() {
  const result = await vercel.authentication.verifyToken({
    token: "<value>",
  });

  // Handle the result
  console.log(result)
}

run();
```

## emailLogin

### Example Usage
